原神简介

在一个被称作“提瓦特”的世界，被神明选中的人将被授予引导元素力量的“神之眼”， 而最终登上天空岛的人会获得神之心， 这些人被称为“原神”。 玩家将扮演“旅行者”与性格各异、能力独特的同伴们一起冒险旅行，并发掘“原神”的真相。 玩家可以选择扮演旅行者中的哥哥或者妹妹，寻找自己失散的唯一亲人。
